To create a superadmin user:

1.  **Open your terminal.**
2.  **Navigate to the `apps/backend` directory** of your project (i.e., `C:\Users\alimi\Desktop\webdev\app-environment-practice\apps\backend`).
3.  **Run the seeder command**:
    ```bash
    npm run seed
    ```
    This script will execute both the `HolidaySeeder` and the `SuperAdminSeeder`. You should see messages in your terminal indicating the seeding process has started and completed. If a superadmin user with the default email already exists, it will be updated; otherwise, a new one will be created.
